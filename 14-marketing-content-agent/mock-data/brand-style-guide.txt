LOGO USAGE
=============================================================================

PRIMARY LOGO:
- Full-color horizontal lockup (preferred)
- Use on white or light backgrounds
- Minimum size: 120px width (digital), 1 inch (print)
- Clear space: Minimum 0.5x the height of the logo on all sides

LOGO VARIATIONS:
1. Primary: Full-color horizontal
2. Stacked: Full-color vertical (for square spaces)
3. Monochrome: Black logo (light backgrounds)
4. Reverse: White logo (dark backgrounds)
5. Icon only: Use when brand is well-established

LOGO DON'TS:
✗ Do not rotate or distort the logo
✗ Do not change colors
✗ Do not add effects (shadows, gradients, outlines)
✗ Do not place on busy backgrounds
✗ Do not use outdated logo versions

COLOR PALETTE
=============================================================================

PRIMARY COLORS:

Brand Blue (Primary)
- HEX: #0066CC
- RGB: 0, 102, 204
- CMYK: 100, 70, 0, 0
- Pantone: 2728 C
- Use: Primary CTAs, headers, brand elements

Slate Gray (Secondary)
- HEX: #2C3E50
- RGB: 44, 62, 80
- CMYK: 81, 68, 50, 48
- Use: Body text, secondary elements

White
- HEX: #FFFFFF
- RGB: 255, 255, 255
- Use: Backgrounds, negative space

SECONDARY COLORS:

Bright Teal (Accent)
- HEX: #00D4AA
- RGB: 0, 212, 170
- Use: Success states, highlights, accents

Coral (Accent)
- HEX: #FF6B6B
- RGB: 255, 107, 107
- Use: Error states, attention, warnings

Light Gray (Neutral)
- HEX: #F5F7FA
- RGB: 245, 247, 250
- Use: Backgrounds, cards, containers

Medium Gray (Neutral)
- HEX: #95A5A6
- RGB: 149, 165, 166
- Use: Borders, dividers, secondary text

COLOR USAGE GUIDELINES:
- 60% Primary colors (Blue, White, Slate)
- 30% Neutral colors (Grays)
- 10% Accent colors (Teal, Coral)

ACCESSIBILITY:
- All text must meet WCAG AA contrast standards (4.5:1)
- Brand Blue on white: ✓ Pass (7.4:1)
- Slate Gray on white: ✓ Pass (12.6:1)
- Medium Gray on white: ✗ Fail for body text (use for large text only)

IMAGERY GUIDELINES
=============================================================================

PHOTOGRAPHY STYLE:
✓ Bright and optimistic
✓ Authentic (real people, not overly staged)
✓ Diverse and inclusive
✓ Focus on people using technology naturally
✓ Light, airy backgrounds
✓ Shallow depth of field for focus
✗ Dark or moody
✗ Stock photo "cheesy" poses
✗ Cluttered or busy

ILLUSTRATION STYLE:
✓ Simple, clean line art
✓ 2-3 colors maximum from brand palette
✓ Modern, geometric shapes
✓ Friendly and approachable
✓ Consistent stroke weight (2-3px)
✗ Complex or detailed illustrations
✗ Cartoon or playful style
✗ 3D or realistic renders

ICONOGRAPHY:
✓ Outlined style (not filled)
✓ 24x24px base size, scalable
✓ 2px stroke weight
✓ Rounded corners (2px radius)
✓ Aligned to pixel grid
✓ Use from consistent icon set (Feather Icons)

IMAGE TREATMENTS:
- Overlay: 40% Brand Blue (#0066CC) with multiply blend
- Gradient overlay: Linear gradient from Brand Blue to Teal
- Duotone: Brand Blue and White
- No filters: Keep images natural and professional

=============================================================================
BRAND VOICE & MESSAGING
=============================================================================

DO:
✓ Use active voice ("We help you..." not "You are helped...")
✓ Be specific with benefits ("Save 10 hours per week")
✓ Show don't tell ("See it in action" not "Trust us")
✓ Use "you" and "your" (customer-focused)
✓ Break up long paragraphs (3-4 sentences max)

DON'T:
✗ Use jargon or buzzwords ("synergy", "paradigm shift")
✗ Make unsubstantiated claims ("Best in the world")
✗ Use all caps for emphasis (use bold or italics)
✗ Overuse exclamation points (one per page max!)
✗ Write in passive voice

TONE VARIATIONS:

Website/Marketing: Confident, inspiring, benefit-focused
Social Media: Conversational, helpful, engaging
Documentation: Clear, instructional, thorough
Support: Empathetic, patient, solution-oriented
Sales: Consultative, value-focused, professional
